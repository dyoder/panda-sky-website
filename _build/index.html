Panda Sky allows you to publish APIs just as you would publish static Web sites to AWS S3 and CloudFront. Your API will run with the same reliability, security, and elasticity as Amazon Web Services!

Quick Start

Initialize your app.

sky init

Define your API.

Edit your api.yaml:

resources:

  greeting:

    path: "/greeting/{name}"
    description: Returns a greeting for {name}.

  methods:

    get:
      method: GET
      signature:
        status: 200

Define a handler.

Add a JavaScript file under lib/sky.js:

module.exports = (data, context, callback) => callback `Hello ${data.name}!`
